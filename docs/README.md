# Team Flow 文档索引

Team Flow 是一个面向小团队的项目协作平台，采用 Spring Boot + React 前后端分离 B/S 架构。

## 文档列表

| 文档 | 说明 |
| --- | --- |
| [MVP 产品与接口设计](./mvp-design.md) | 项目概述、功能范围、业务流程、模块拆分、迭代计划 |
| [REST API 设计文档](./api-design.md) | 接口统一约定、认证、项目、成员、任务、评论、统计接口 |
| [数据库设计文档](./database-design.md) | MySQL 表结构、字段说明、枚举、索引、SQL 草案 |

## 推荐阅读顺序

1. 先读 `mvp-design.md`，确定项目做什么、不做什么。
2. 再读 `database-design.md`，确定后端实体和表结构。
3. 最后读 `api-design.md`，确定前后端接口契约。
