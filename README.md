# Team Flow

Team Flow 是一个面向小团队的项目协作平台，支持用户创建项目、邀请成员、分配任务、拖拽任务状态、评论沟通和查看项目进度。

## 技术方向

- 后端：Spring Boot
- 前端：React
- 架构：前后端分离 B/S 架构
- 数据库：MySQL 8.x

## MVP 功能

- 用户注册、登录、获取当前用户信息
- 创建、查看、编辑、删除项目
- 邀请成员加入项目，维护项目成员列表
- 创建、分配、编辑、删除任务
- 通过看板拖拽任务状态和排序
- 在任务详情中发表评论
- 查看项目任务进度和成员任务统计
